# C2C-Ecommerce
A simple C2C (Customer to Customer) e-commerce platform implemented in Java. The platform allows users to register, log in, list products, browse products, add products to a shopping cart, purchase products, and leave reviews. It also includes a simulated payment system with transaction locking to prevent data inconsistencies during concurrent transactions.
